Japanese-English Phonetic Assistant

Translate Japanese sentences into English with phonetics and pronunciation support!

📌 What is this extension?

This Chrome Extension helps you translate any Japanese sentence into English with accurate translation and phonetic (romaji) transcription using OpenAI's language capabilities.

✨ Bonus: If you're learning English, the extension also provides the IPA (International Phonetic Alphabet) transcription of the English translation – helping you pronounce English words correctly.

✨ Key Features

  • 🔍 Right-click any Japanese sentence to instantly see the English translation.
  • 🗣️ Includes romaji (Japanese phonetic transcription).
  • 🔤 Includes English IPA phonetic transcription (optional).
  • ⚡ Fast and seamless – works directly on the page you're viewing.
  • 💬 Powered by OpenAI for natural, high-quality translations.

🎯 Who is it for?

  • 🇯🇵 Japanese users learning English
  • 🇬🇧 English speakers reading Japanese content
  • 🎓 ESL (English as a Second Language) learners who want to improve pronunciation
  • 📖 Language learners who want dual understanding

🚀 How to use

  1. Install the extension from Chrome Web Store.
  2. Visit any webpage with Japanese text.
  3. Select (highlight) a Japanese sentence.
  4. Right-click and choose “Translate with OpenAI”.
  5. See English translation + romaji + (optionally) English phonetics!

📌 この拡張機能とは？

このChrome拡張機能は、Webページ上の日本語の文章を選択して右クリックするだけで、英語の翻訳とローマ字での日本語発音を表示してくれるツールです。

✨さらに: 英語学習者のために、翻訳された英語の発音記号（IPA）も表示することが可能です。正しい発音の習得に役立ちます。

✨ 主な機能

  • 🔍 日本語の文を選択して右クリックすると、即座に英訳が表示されます。
  • 🗣️ ローマ字（日本語の発音）を一緒に表示。
  • 🔤 英語のIPA（発音記号）も表示可能。
  • ⚡ Webページ上でそのまま動作、スムーズな使い心地。
  • 💬 OpenAIの言語AIによる高品質な翻訳。

🎯 対象ユーザー

  • 🇯🇵 英語を学ぶ日本人ユーザー
  • 🇬🇧 日本語を学ぶ英語圏のユーザー
  • 🎓 正しい英語の発音を知りたい英語学習者
  • 📖 日英バイリンガルで読みたい全ての学習者

🚀 使い方

  1. Chromeウェブストアから拡張機能をインストール
  2. 日本語の文章があるWebページにアクセス
  3. 翻訳したい文を選択（ハイライト）
  4. 右クリックして「OpenAIで翻訳」を選択
  5. 翻訳・ローマ字・英語発音（IPA）が表示されます！
